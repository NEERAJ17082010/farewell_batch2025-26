# Batch of 2025-26 - Farewell Site

An interactive farewell website for the Batch of 2025-26 with password-protected profiles and photo memories.

## Features

- Minimal Dark Design: Black screen with subtle dot pattern background
- Profile System: Pre-created profiles for all 16 students with unique passwords
- Password Protection: Each profile requires its unique password to access
- Photo Comparison: Side-by-side childhood and current photos
- Personal Notes: Unique custom messages for each person
- Photo Upload: Easy way to add real photos later
- Thank You Page: Common page with bold, italic, glowing "THANK YOU" text

## How to Use

### 1. Opening the Site
Open `index.html` in any modern web browser.

### 2. Accessing Your Profile
1. Enter your name (case-sensitive, e.g., "Bhanushree")
2. Click "Find Profile"
3. Enter your unique password
4. View your memory page

### 3. Adding Photos
On the memory page, use the "Add Your Photos" section to:
- Upload your childhood photo
- Upload your current photo

### 4. Viewing Thank You Page
Click "View Thank You" button on the memory page to see the common thank you page.

## Site Flow

1. Landing Page: Enter your name
2. Profile Page: Enter password to unlock
3. Memory Page: View photos and personal note, upload photos
4. Thank You Page: Bold, italic, glowing "THANK YOU" text

## Customization

- Edit `script.js` to change personal notes in the `personalNotes` object
- Edit `style.css` to change colors, fonts, or styling
- All photos are stored in browser's localStorage

## Notes

- Search is case-sensitive
- Photos are stored in the browser's localStorage (limited space)
- Each person gets their own unique password
- The design is unisexual and minimal

Works best in:
